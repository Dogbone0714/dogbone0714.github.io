樹莓派如何輸入中文字？

發表於 2021-07-12 分類於 樹莓派 閱讀次數： Valine：

拿到樹莓派的你是否發現，設定中預設的中文，並不能打中文？？沒關係，這篇文教你如何安裝中文輸入法及設定中文！

  1. 打開 Terminal

  2. 安裝SCIM輸入法

    sudo apt-get install scim scim-tables-zh

  3. 查看SCIM輸入法的切換查看及設定
    (樹莓派LOGO→偏好設定→SCIM輸入法框架)

(功能切換的快速鍵)

開啟鍵盤功能：Control + Space

設定輸入法切換：

  1. 點下一種輸入法旁的三個點點

  2. 點選畫面中的三個點點

  3. 再按下你想要設定切換的按鍵，我這邊是設定 左邊的shift。

  4. 最後按下「+」鍵以及確認即可。

  5. 之後按下畫面中的勾勾即可作用。
