打造屬於自己的縮網址服務！

發表於 2024-10-26 更新於 2024-10-27 分類於 軟體介紹 閱讀次數： Valine：
1.6k

想要打造專屬的縮網址系統，不再依賴第三方服務？YOURLS (Your Own URL Shortener) 是一款免費開源的縮網址解決方案，讓你能輕鬆在自己的主機上架設獨立的縮網址服務，不僅支持自訂短網址、還提供詳細的點擊分析，方便追蹤流量來源。本篇文章將帶你了解 YOURLS 的特色、安裝方法及使用技巧，讓你能充分掌控自己的縮網址系統，提升分享效果和網站管理效率。

YOURLS

YOURLS 是什麼？

YOURLS，全名為 Your Own URL Shortener，是一款免費開源的縮網址工具。與其他公共縮網址平台（如 bit.ly 或 tinyurl）不同，YOURLS 允許使用者將縮網址服務安裝在自己的伺服器上，提供了完全的掌控權和更高的隱私性。

YOURLS 的主要特色

  1. 完全掌控： YOURLS 讓你可以自定義短網址的結構和格式，例如根據品牌或個人需求來選擇字串。這對企業和網站經營者特別有吸引力，因為它不僅增強了品牌識別度，也保障了短網址的長期有效性。

  2. 詳細的流量分析： YOURLS 內建了點擊統計功能，可追蹤短網址的訪問次數、來源、使用的裝置等詳細資訊，便於流量分析和行銷決策。

  3. 豐富的插件支持： YOURLS 支援各種插件，可以加入如密碼保護、過濾訪問 IP、設置短網址到期時間等功能，使其更加靈活且適應各種需求。

  4. 開源、免費、社群支持： YOURLS 是開源軟體，因此完全免費，且具備社群支持。這意味著使用者可以享有穩定的版本更新，也能與開發者一起討論需求和改進。

安裝步驟

Step 1 需求準備

在開始之前，確保你的伺服器具備以下條件：

  • 支援 PHP 版本 7.4 或更高。

  • 支援 MySQL 或 MariaDB 資料庫。

  • 自行註冊並擁有一個網域（如 myshort.link）。

Step 2 下載 YOURLS

  1. 進入 YOURLS 官方網站 或 YOURLS 的 GitHub 頁面 下載最新版本的 YOURLS 壓縮檔案。

  2. 將壓縮檔案解壓縮，並上傳至伺服器上的目標資料夾。

Step 3 建立資料庫

  1. 進入你的網站主機控制面板（如 cPanel）。

  2. 建立一個新的 MySQL 資料庫，並記下資料庫名稱、使用者名稱和密碼，之後會在 YOURLS 設定中使用到。

Step 4 配置 YOURLS 設定檔案

  1. 在 YOURLS 的根目錄中找到 user/config-sample.php，將其重新命名為 config.php。

  2. 開啟 config.php，修改以下主要參數：

    • YOURLS_DB_USER：填入資料庫的使用者名稱。
    • YOURLS_DB_PASS：填入資料庫的密碼。
    • YOURLS_DB_NAME：填入資料庫名稱。
    • YOURLS_SITE：填入你的 YOURLS 網站 URL（如 https://myshort.link）。
    • YOURLS_HOURS_OFFSET：設定伺服器時區。
    • YOURLS_LANG：設定 YOURLS 的語言（預設為英文，可設定為中文等其他語言）。
    • YOURLS_UNIQUE_URLS：設為 true 時可避免重複短網址生成。
    • YOURLS_PRIVATE：設為 true 可使 YOURLS 成為私人縮網址服務，僅允許認證用戶使用。

  3. 設定管理員帳戶：

    • 在 YOURLS 內 YOURLS_USER 中設定管理員帳號和密碼，確保你的帳戶安全。

Step 5 開始安裝

  1. 確認所有設定都正確後，打開瀏覽器並前往 https://你的網域/admin（如 https://myshort.link/admin）。
  2. 依照提示完成安裝流程，YOURLS 會自動創建需要的資料表。
  3. 完成後，你可以登入 YOURLS 管理介面，開始建立縮網址！

Step 6 測試 YOURLS 是否正常運行

進入 YOURLS 管理介面，嘗試建立一個縮網址，並確認是否能成功轉向到目標網址。

恭喜完成！這樣你就擁有屬於自己的縮網址服務，不用擔心安全性的問題~~
