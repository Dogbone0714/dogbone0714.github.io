文創系其實不文？4年我如何跨足數位與產品開發

發表於 2025-07-15 分類於 經驗分享 閱讀次數： Valine：
1.5k

「你是文創系的喔？那你會畫畫設計嗎？」
「文創系是不是每天都在剪紙寫詩做陶土？」
「欸所以你們畢業以後是要去文創園區擺攤嗎？」

身為文創系學生，我早就習慣這些問句。
但其實，這四年我做的事，比很多人想像的「文創」還要技術、還要複雜。

先來介紹我的背景：

高中三年的文組生，卻特殊選才至理工大學 而且還是資工系

資訊工程學系一年半的讀書經驗

跨轉三年半的文化創意與數位行銷學系

順利畢業 ( 現在在這

🎓 文創系的底子：我們學的是「整合與轉換」

在國立聯合大學(石虎大學)文化創意與數位行銷學系，我們的課程不只有美學和設計，也包含：

  • 敘事策略與內容企劃 (可以選修企畫書撰寫)
  • 社會學與文化研究(還有社會心理學也是必修zzZ)
  • 數位行銷與媒體操作
  • UIUX與創意簡報
  • Python 程式設計、網站前端(HTML & CSS 但我個人愛好WordPress)、互動遊戲程式應用

學的是什麼？學的是如何理解人、轉換需求，並把想法變成內容或產品。

🚀 我怎麼從「文創系學生」走向「產品實作」

我用 4 年時間，讓自己成為一個跨界使用者與設計者，也真的做出了不少東西(雖然很微小)。

1️⃣ 開始於學生自治：從紙本到數位協作

在學生議會與學生會任職期間，我們過去的會議記錄、資料傳遞都靠紙本和口頭。
我開始導入 Google Workspace、Notion、Lark，整合內部行政與傳承系統，也寫了第一份會議資料自動整理腳本。

👉 從溝通問題 → 設計流程 → 導入工具 → 讓資訊流動起來
這就是我學到的「數位轉型的原點」。

2️⃣ 創業實驗：《繁花之聲》手遊企劃與開發

我的畢業專題即是設計一款以客家文化為背景的問答策略遊戲《繁花之聲》，針對 10～29 歲年輕族群推廣文化知識。

  • 從世界觀、關卡設計到知識題庫資料整理
  • 用 GameSalad 開發 MVP
  • 投入市場分析與使用者測試

👉 這個專案讓我學會：文化不只是創意，也可以成為有邏輯、有產品化潛力的素材。

3️⃣ Side Project：《聯大助手 Open NUU》

並且我還主導了非官方校園數位平台《聯大助手 OPEN NUU》的設計與構建：

  • 整合選課資訊、即時公告、教室位置指引
  • 架設網站、部署伺服器
  • 設計操作介面、撰寫使用說明

後來也近一步推動開源協作的《Open NUU》計畫，嘗試讓學生用開源文化來改善校園服務體驗。

👉 文創系也可以開發服務、維運工具、帶團隊設計產品。

4️⃣ 幫長照機構數位轉型

我最近協助「大 O 居家長照機構」設計居服員用的資訊整合工具，從 LINE 使用習慣出發，設計一套無痛導入的資訊平台，改善紙本記錄與混亂通訊問題。

👉 設計的核心從不是工具，而是「照顧人怎麼用、會不會用、用了之後會不會更輕鬆」。

📣 我想說的是：文創系不是你以為的那樣

我們不是只是設計漂亮的東西、辦展覽、做創意小物。
我們其實是——

🎯 看懂文化底層需求的人
🧠 懂得轉化為故事、產品、服務的人
🛠 願意實作、用工具解決問題的人

而這些能力，是所有領域最需要的跨域力。

📬 想給學弟妹與未來的自己一句話

如果你是文創系學生，不用怕別人說你不夠專業。
專業是靠實作出來的，你可以成為設計師、開發者、產品人、PM、創業者——只要你願意打破框架。

而我，會持續在這條路上前進，讓更多人知道：

文創系，不文。我們很硬核。
