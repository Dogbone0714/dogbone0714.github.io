每次搬新家都會忘記的 PicGo 設定

發表於 2026-04-05 更新於 2026-04-06 分類於 教學 閱讀次數： Valine：
541

對於一個習慣用 Markdown 寫作、經營技術部落格的人來說，PicGo 簡直是生命線。但在更換電腦、重新安裝系統（也就是所謂的「搬新家」）時，最崩潰的莫過於要重新設定那一串長長的圖床參數。

每次都要翻舊文件、進雲端後台找 Token？這篇文章幫你整理好那些「每次搬家都會忘、每次都要重查」的核心設定，建議直接收藏，下次搬家看這篇就夠了。

常用 Typora 寫文章的人，一定少不了使用 PicGo，但每次換電腦時，就要重新設定 config 值，是否讓你感到頭痛，每次都要重新去找尋之前設定的數值，這篇文章將告訴你全攻略，包含設定檔都留給你，也提醒我自己。
